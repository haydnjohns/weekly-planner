Documentation & Shortcuts

Create Tasks

  • Double-click empty grid: Create default 1hr task.
  • Double-click + Drag down: Draw a custom length task visually.

Select Tasks

  • Click & Drag empty grid: Lasso selection.
    • Left → Right: Select tasks fully inside box.
    • Right → Left: Select tasks intersecting box.
  • Ctrl+Click: Add/remove single task.
  • Ctrl+A: Select all tasks.
  • Esc: Deselect all.

Edit Tasks

  • Double-click Title: Edit name directly inline.
  • Drag / Resize: Applies to all selected tasks simultaneously.
  • Ctrl+C / Ctrl+V: Copy and Paste selected tasks.
  • Backspace/ Del: Delete selected tasks.
  • Esc: Cancel.

Categories & View

  • Select Tasks + Click Category: Instantly change color/category of selected items.
  • Click Category (None Selected): Edit category details.
  • Drag Time Column (Left side): Zoom interface vertically.
  • Ctrl+Z / Ctrl+Y: Undo / Redo.
