## Description

Ceci n'est pas un calendrier.

This app is a **desktop-focused weekly planner** that prioritizes ideation. It helps you efficiently organize and visualize your week, focusing on planning and productivity rather than rigid scheduling.  

## Features

- **Shuffling and rearranging**: Move events around freely to test priorities, timing, and layout without the friction of a traditional calendar app.
- **Drag and drop**: Intuitively click and drag events to new slots, making adjustments fast and seamless.  
- **Quick edit in place**: Edit event details directly on the grid without opening separate dialogs.  
- **Color-coded categories**: Assign colors to different types of events for instant visual organisation.  
- **Keyboard shortcuts**: Navigate, create, and modify events efficiently without relying solely on the mouse.  
- **Intuitive multi-select**: Select multiple events, copy, edit and reposition them to plan complex weeks quickly.  
- **Print wizard**: Export a formatted version of your week for printing or offline use.
